Programa que permite gestionar un restaurante con diferentes tipos de platos, y calcular el precio con iva, recargos y gestionar las propinas. 
Características:
  * 3 tipos de platos: Entradas, Platos Fuertes y Postres
  * Cálculo automático de precios con IVA (21%)
  * Recargo especial para platos fuertes (+15%)
  * Sistema de descuentos para postres
  * Gestión de propinas con sobrecarga de métodos
  * Capacidad máxima de 10 platos por pedido
  * Validación de precios mediante excepciones personalizadas

**Funcionalidades
Gestión de Platos
  * Entrada: Precio base + IVA (21%)
  * Plato Fuerte: Precio base + Recargo (15%) + IVA (21%)
  * Postre: Precio base + IVA (21%) + Descuento opcional

Sistema de Pedidos
  * Agregar hasta 10 platos diferentes al pedido
  * Visualización del pedido completo con precios detallados
  * Cálculo de subtotal (sin propina)
  * Cálculo de total (con propina)
  * Control de capacidad máxima del pedido
